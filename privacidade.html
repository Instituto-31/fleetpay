Política de Privacidade

Levamos a sério a tua privacidade e a dos teus motoristas. Esta política explica que dados recolhemos, porquê, durante quanto tempo, e quais os teus direitos. Cumpre o RGPD (Regulamento UE 2016/679).

1. Quem é o responsável pelo tratamento

O Instituto 31, NIPC 517069234, com sede em Portugal, é responsável pelo tratamento dos dados pessoais relativos à tua conta FleetPay (operador) e contactos comerciais.

Para os dados dos teus motoristas e dados operacionais (viagens, pagamentos, viaturas), tu (operador) és o responsável pelo tratamento e o FleetPay actua como subcontratante nos termos do art. 28.º RGPD.

2. Que dados tratamos

CategoriaExemplosFinalidade
Conta de operador Nome, email, telefone, NIPC, password (hash) Criar e gerir a conta; comunicar contigo
Empresa Nome, NIPC, morada, licença TVDE, logo Operação da plataforma e documentos legais
Motoristas Nome, NIF, IBAN, contactos, documentos (CC, Carta, TVDE) Pagamentos, contratos, compliance TVDE
Viaturas Matrícula, marca, IUC, seguro, manutenção Gestão de frota
Operacionais Viagens (Bolt/Uber), pagamentos, despesas, comunicações Cálculo de recibos, histórico, dossier técnico
Técnicos IP, browser, registos de acesso, cookies estritamente necessários Segurança, prevenção de fraude

3. Base legal

  • Execução do contrato (art. 6.º/1/b RGPD) — para te prestar o serviço subscrito
  • Obrigações legais (art. 6.º/1/c RGPD) — facturação, retenção fiscal, cumprimento TVDE
  • Interesse legítimo (art. 6.º/1/f RGPD) — prevenção de fraude, segurança e melhoria do serviço
  • Consentimento (art. 6.º/1/a RGPD) — para emails de marketing (que podes recusar a qualquer momento)

4. Subcontratantes

Para entregar o serviço usamos os seguintes subcontratantes, todos com garantias adequadas de protecção de dados:

  • Supabase (base de dados e autenticação) — infraestrutura na União Europeia (Frankfurt, AWS)
  • GitHub Pages (frontend estático)
  • Resend (envio de emails transaccionais)
  • Bolt e Uber (apenas para integração API oficial; os dados que recolhemos das tuas viagens vêm destas plataformas, com base nas tuas credenciais)

5. Localização dos dados

Os dados são armazenados em infraestrutura na União Europeia (servidores AWS Frankfurt geridos pela Supabase). Não fazemos transferências para fora do EEE excepto quando indispensável para o serviço (ex.: APIs Bolt/Uber), com salvaguardas contratuais adequadas.

6. Prazos de conservação

  • Dados de conta activa: enquanto a conta estiver activa
  • Após cancelamento: 30 dias úteis para eliminação, salvo retenção legal obrigatória
  • Dados financeiros (recibos, facturas): 10 anos (art. 123.º CIRC)
  • Dados TVDE (contratos, fiscalização): conforme legislação aplicável (≥5 anos)
  • Registos técnicos de acesso: até 6 meses

7. Os teus direitos

Como titular dos dados, tens direito a:

  • Acesso — saber que dados temos sobre ti
  • Rectificação — corrigir dados incorrectos
  • Apagamento ("direito ao esquecimento") — pedir eliminação dos dados
  • Limitação do tratamento
  • Portabilidade — receber os dados em formato estruturado (CSV/JSON)
  • Oposição — recusar o tratamento para certas finalidades (ex.: marketing)
  • Não decisões automatizadas — não tomamos decisões com efeitos jurídicos baseadas exclusivamente em tratamento automatizado

Para exercer estes direitos, envia email para geral@fleetpay.pt. Respondemos em 30 dias.

8. Cookies

O FleetPay usa apenas cookies estritamente necessários ao funcionamento (sessão de login, preferências de tema). Não usamos cookies de marketing ou tracking de terceiros.

9. Segurança

Aplicamos medidas técnicas e organizativas adequadas:

  • Encriptação em trânsito (TLS 1.3) e em repouso
  • Row Level Security (RLS) ao nível da base de dados — cada operador só vê os seus dados
  • Passwords armazenadas com hash bcrypt
  • Backup automático diário
  • Acesso aos sistemas restrito e auditável

10. Reclamações

Se considerares que o tratamento dos teus dados viola o RGPD, podes apresentar reclamação à autoridade de controlo competente:

  • Comissão Nacional de Protecção de Dados (CNPD)
  • Web: www.cnpd.pt

11. Contacto

Encarregado de Protecção de Dados (DPO): geral@fleetpay.pt
