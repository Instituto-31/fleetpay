# 🎬 Vídeo Motorista — 30 segundos

**Objectivo:** Mostrar ao motorista TVDE como usar a PWA dele.
**Ferramenta:** Loom + telemóvel (gravação ecrã iPhone/Android — não Loom).
**Onde vai:** Anexo ao email/WhatsApp quando o operador adiciona um motorista novo.

---

## Setup

1. **Telemóvel** com `fleetpay.pt` instalado como PWA
2. Login como motorista demo (joana@demo.fleetpay.pt — se a demo permitir signin sem password, senão usa magic link)
3. **Gravar ecrã do telemóvel** (não Loom):
   - iOS: Centro de Controlo → Gravar Ecrã
   - Android: deslizar baixo 2x → Gravação de Ecrã
4. Pega no telemóvel na mão (não na mesa) — mais natural
5. Voz por cima depois (Loom permite voiceover de ficheiro carregado)

---

## Script (30 segundos)

| Tempo | Ecrã telemóvel | Acção | Narração |
|---|---|---|---|
| 0-5s | Home FleetPay (motorista) | Mostra última semana com valor grande dourado | "Esta é a tua vista no FleetPay. Aqui vês o que o operador te vai transferir esta semana." |
| 5-12s | Toca em banner laranja "🔔 Validar valor" | Aparece detalhe + 2 botões | "Antes da transferência, validas o valor. Se concordas, ✓ Concordo." |
| 12-18s | Clica ✓ Concordo | Estado muda para "✓ Pronto p/ transferir" | "Se algo não bate certo, ⚠️ Não concordo e escreves o motivo." |
| 18-24s | Vai a tab Despesas | Mostra PRIO + Via Verde da semana | "Vês as despesas que foram descontadas — combustível, portagens, slot." |
| 24-30s | Recibo PDF aberto | "Quando o operador transferir, recebes recibo PDF para a tua contabilidade." | |
